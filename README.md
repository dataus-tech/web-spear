# web-spear

## 테스트 방법

```
1. git clone https://github.com/dataus-tech/web-spear.git
2. cd web-spear
3. npm install
4. npm run build
5. chrome://extensions/ 으로 이동
6. 압축해제된 확장 프로그램을 로드합니다. 버튼 클릭
7. 4에서 build된 폴더를 선택
```
